🚨 Smart Safety Network

👥 Team: Hack Attackers

College: Government Women Engineering College (GWEC), Ajmer

---

📌 Problem Statement

Ensuring safety in real-time is a major challenge due to delayed emergency response and lack of immediate communication systems.

---

💡 Solution Overview

The Smart Safety Network is an AI-powered emergency response system that detects distress situations and instantly sends alerts to the nearest police station, family members, and nearby volunteers.

---

⚙️ Key Features

- 🚨 SOS alert within 3 seconds
- 📍 Real-time location tracking
- 👮 Nearest police station auto-alert
- 📡 Works even without internet (LoRaWAN backup)
- 🔔 Notifications to family & volunteers
- 🧠 AI-based distress detection

---

🧠 How It Works

1. Distress is detected using phone sensors
2. SOS alert is automatically triggered
3. Alert is sent to the nearest police station
4. Family & volunteers receive notifications
5. Help reaches the victim quickly

---

🌍 Real-World Application

This software can be installed directly on smartphones.

- In new phones, it can come pre-installed
- In existing phones, it can be delivered through system updates

It uses sensors already available in smartphones, eliminating the need for additional hardware.
